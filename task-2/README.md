# 练习二：hello vue!

## 使用说明

1、使用`git clone ...`将项目clone到本地的文件夹中，

2、在当前文件夹中执行`npm i`命令行在本地安装依赖包；

3、执行`webpack`,然后打开page.html，你就可以看到Hello Vue ! (以及[vue.js广官方文档]()上的几个例子)

---
